# Keff Mobile (React Native + TypeScript)

Expense tracking and CRM interface for professionals.

## Setup

```bash
npx react-native@latest init KeffMobile --template react-native-template-typescript
cd KeffMobile
npm install
npx react-native run-ios # or run-android
```

## Stack

- React Native
- TypeScript
- React Navigation
- Zustand (state management)
- React Query (API caching)
