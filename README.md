# Keff Mobile App

Expense manager + CRM for finance advisors, economic consultants, and professionals.

## Tech Stack

- **Frontend**: React Native + TypeScript
- **Backend**: Node.js + Express + TypeScript
- **Database**: PostgreSQL
- **Auth**: JWT + OAuth2
- **Infra**: Docker, Docker Compose

## Project Structure

```
keff/
├── mobile/          # React Native app
├── server/          # Node.js backend API
├── docker-compose.yml
├── .env.example
└── README.md
```
